# Images Directory

This directory contains all images used on the SignFire website.

## Required Images

Please add the following image files to this directory:

### 1. **logo.png**
- Your SignFire logo
- Recommended size: 120px height (width proportional)
- Format: PNG with transparent background preferred
- Used in: Hero section header

### 2. **painting1.jpg**
- Gallery image - Canvas Mandala
- Recommended size: 800x800px or larger
- Format: JPG
- Used in: Gallery section

### 3. **stone1.jpg**
- Gallery image - Painted Stone/Boulder
- Recommended size: 800x800px or larger
- Format: JPG
- Used in: Gallery section

### 4. **painting2.jpg**
- Gallery image - Canvas Mandala
- Recommended size: 800x800px or larger
- Format: JPG
- Used in: Gallery section

## Image Guidelines

- Use high-quality images that showcase your artwork
- Optimize images for web (compress JPGs to 80-90% quality)
- Ensure good lighting and clear visibility of the mandalas
- For gallery images, consider using square crops for consistency
- Maximum file size recommendation: 500KB per image (after optimization)

## Adding Images

To add images to this directory:

1. Place your image files in this folder
2. Ensure they match the filenames listed above
3. Commit and push the changes:
   ```bash
   git add images/
   git commit -m "Add website images"
   git push
   ```

Your website will automatically update once pushed to the main branch.
